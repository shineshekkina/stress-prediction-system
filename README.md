# Stress Prediction with Encouragement

##  Overview
This project is a web-based application that predicts user stress levels and provides motivational encouragement along with Bible verses.

##  Technologies Used
- Python
- Django
- HTML
- CSS

##  Features
- Predicts stress levels based on user input
- Displays encouraging messages
- Provides Bible verses for motivation
- Simple and user-friendly interface

##  Purpose
This project aims to promote mental well-being by combining stress analysis with positive reinforcement and spiritual encouragement.

##  How to Run
1. Clone the repository  
2. Install required packages  
3. Run the server:
   python manage.py runserver  
4. Open browser at:
   http://127.0.0.1:8000/
